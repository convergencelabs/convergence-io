Convergence is more than just the core platform. We've also built an entire community focused on building collaborative applications.

Community Resources

Convergence Labs depends on a vibrant community of users to create awesome applications that users love. We contribute ideas, tutorials, open source software, and make ourselves accessible to the community to make that as easy as possible. We love it when our users share ideas, show off things they have built, help other users, and contribute to our growing set of open source components. Here are some place where you can participate in the Convergence Community:


  • Convergence Labs Blog: We share thoughts and best practices around developing applications with rich collaboration features.
  • Convergence Labs Git Hub Organization: Find code examples, bindings open source toolkits, and other helpful code.
  • Convergence NPM Organization: Find NPM modules that simplfiy using Convergence to build awesome collaborative apps.
  • Convergence Discourse Forum: A great place to ask questions and discuss development. This is the best place to ask for support if you are not on a paid plan.
